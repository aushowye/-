# 福建题库在线刷题系统

一个基于纯前端的在线刷题系统，支持单选题、多选题、判断题和案例题。

## 📊 题库统计

- **总题数**: 1535 道
- **公共部分**: 698 道
- **专业部分**: 837 道

### 题型分布
- 单选题: 528 道
- 多选题: 356 道
- 判断题: 606 道
- 案例题: 45 道

## 🚀 功能特性

- ✅ 单题显示，逐题答题
- ✅ 实时答案反馈
- ✅ 进度追踪和统计
- ✅ 按分类、题型筛选
- ✅ 响应式设计，支持手机
- ✅ 纯前端，无需后端

## 📁 文件说明

- `exam.html` - 刷题网页主文件
- `questions.json` - 题库数据文件
- `parse_questions.py` - 题库解析脚本

## 💻 本地运行

1. 下载所有文件
2. 双击打开 `exam.html`
3. 或使用本地服务器：
   ```bash
   python -m http.server 8000
   ```
   然后访问 http://localhost:8000/exam.html
